Getting Started with Jacobi Method Toolbox

The Jacobi Method Toolbox provides a robust implementation of the Jacobi method for solving systems of linear equations. It is designed for both standard MATLAB® arrays and distributed arrays, making it a powerful tool for numerical linear algebra.
The core of the toolbox is the jacobi function. It attemps to solve the linear system for the unknown vector . The matrix A must be squared and should be strictly row diagonally-dominant to guarantee the convergence of the Jacobi Method.
